Let's Print some more Strings

Let's write our first line of code in the terminal

print "Hello World!"

This is usually the first line that all programmers write to see how to print text into the terminal. Your terminal window should return the following code:

Hello World!=> nil

Now, let's print numbers and text. To print text into the terminal, you must include the text within "quotes", oposite to numbers, which it is not required to be inside quotes. Write the following code:

print "Hello, student!"
print 100

That code should show this:

Hello, student!=> nil
100=> nil

We must be able to concatenate strings

puts "Hello " + "Leia"

Your terminal window should return the following code:

Hello Leia
=> nil

Remember to add a space after the word "Hello" or before "Leia", to prevent the string from having a joint string

Now, let's print a paragraph with multiple lines 

 puts << END
this is a here doc.
displays multi line text
displays multi line text.
displays multi line text
displays multi line text
END

That code should show this:

this is a here doc.
displays multi line text
displays multi line text.
displays multi line text
displays multi line text

Other methods for printing

There is p and puts

Use puts

puts "Hello Wolrd!"

This is what the terminal prints:

Hello Wolrd!
=> nil

As you can see, the nil shows up in the line below

Use p

p "Hello Wolrd!"

This is what the terminal prints:

"Hello World!"
=> "Hello World!"

As you can see, p prints out the string (text) inside quotes
